# Temple Services (HTML/CSS/JS)

A mobile-first, devotional-themed web app to manage and book temple services. No backend required; data persists in browser localStorage.

## Pages
- `index.html`: Landing
- `admin-login.html`: Admin sign in
- `admin-dashboard.html`: Add/list temples
- `admin-temple.html?id=...`: Manage services for a temple
- `user-login.html`: Devotee sign in
- `register.html`: Devotee registration
- `temples.html`: Search/browse temples
- `temple.html?id=...`: Temple details + booking
- `bookings.html`: User booking history

## Run
- Open any page directly in a browser, or serve the folder with a static server
- Default admin login: `admin@temple.com` / `admin123`

## Design
- Devotional palette (saffron, gold, deep maroon)
- Large tap targets, responsive grid, accessible forms

## Resetting Data
- Clear localStorage for the site to reset sample data and accounts
