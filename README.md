# Temple Services (HTML/CSS/JS)

A simple responsive web app to manage and book temple services. No backend required; data persists in browser localStorage.

## Open in Visual Studio/VS Code
- Extract the ZIP you downloaded
- File > Open Folder... and select the extracted folder
- Open `index.html` in a Live Server or your browser

## Quick Start (local)
1. Double-click `index.html` to open in your browser
   - Or serve the folder with any static server
2. Default admin login: `admin@temple.com` / `admin123`

## Features
- Admin
  - Add/delete temples
  - Add/delete services with amounts per temple
- User
  - Register/login
  - Search temples by name/location
  - View temple details and services
  - Book services by date and time
  - View and cancel booking history

## Files
- `index.html`: App shell and UI
- `styles.css`: Responsive styles
- `app.js`: App logic with localStorage persistence

## Resetting Data
- Clear site data (localStorage) in your browser to reset the app to defaults.
